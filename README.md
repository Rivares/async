# async

## Utility for transferinf UDP packets

**Features:**

* client
* server

> client send packet. server receive packet.
___

## Options

* `not options`   *to future to make ip and port (\"path\" ... \"path\")*

___

## Work with program

1. Edit ip address and port
```shell
nano ./client_/client.cpp
nano ./server_/server.cpp
```

2. Build
```shell
cmake
make
```

3. Run
```shell
./client_/client
./server_/server
```

4. Stop
```shell
top
kill PID
```
